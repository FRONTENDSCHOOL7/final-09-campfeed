## 5. 구현 기능

  - 게시글 등록/삭제/신고

  - 이미지 최대 3장 업로드

  - 댓글 등록
  - 댓글 삭제
- 👤 Profile
  - 개인 프로필
  - 프로필 수정
  - 팔로우/언팔로우
  - 팔로우/팔로잉 리스트
  - 로그아웃
- 🔎 Search
  - 유저 검색
- ❤️ Like
  - 게시물 좋아요 및 취소
- 🏕️ Campsite
  - 캠핑장 등록/예약
